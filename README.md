# Cloudwork

Cloudwork is a basic project that I made for improve my iOS skills.

## Requirements

```
iOS 13+
Xcode 11
Swift 5
UIKit
```

## API

```
The app uses an API developed with PHP and MySQL
```

* Alamofire
* Kingfisher
* RKDropdownAlert
* SZTextView
